GWcode SyntaxHighlighter with SyntaxHighlighter v2.1.382

First, make sure you include these javascript files:

<script type="text/javascript" src="sh2/scripts/shCore.js"></script>
<script type="text/javascript" src="sh2/scripts/shBrushEE.js"></script>

Second, include the CSS files (you can combine them as 1 css file if you'd like).
The 3rd css file adds the colors for EE code:

<link type="text/css" rel="stylesheet" href="sh2/styles/shCore.css" />
<link type="text/css" rel="stylesheet" href="sh2/styles/shThemeDefault.css" />
<link type="text/css" rel="stylesheet" href="sh2/styles/shEE.css" />

Then, include some javascript to start highlighting code:

<script type="text/javascript">
SyntaxHighlighter.config.strings.alert = '';
SyntaxHighlighter.defaults['auto-links'] = false;
SyntaxHighlighter.config.clipboardSwf = 'sh2/scripts/clipboard.swf';
SyntaxHighlighter.all();
</script>

And this will be the output:

{exp:channel:entries channel="news" limit="1"}
	<h1>{title}</h1>
	<p>Some HTML code</p>

	{exp:pluginname param="value"}{variable-wrapped-in-plugin}{/exp:pluginname}
	<div id="divid">Some more HTML code</div>
	{variable_pair}something in between{/variable_pair}
{/exp:channel:entries}
